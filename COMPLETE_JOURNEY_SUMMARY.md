## User Experience Flow

### 1. Discovery (Home Page)
User lands on home page → Sees transformation promise → Clicks "Start Your Journey" or "View Timeline"

### 2. Journey Selection
User views all 15 stages → Selects a stage → Enters stage detail page

### 3. Stage Completion Flow
1. **Overview**: Read stage description, transformation goal, learning objectives
2. **Pre-Assessment**: Answer 4-option question about current state
3. **Orientation**: Understand why this stage matters, see sub-stages
4. **Activity**: Follow step-by-step instructions with checklists
5. **Reflection**: Think about learnings and challenges
6. **Post-Assessment**: Commit to specific actions
7. **Complete**: See transformation score and next steps

### 4. Alternative Views
- **Presentation Mode**: Full-screen slides for teaching/presenting
- **Timeline**: See entire journey at a glance
- **Roadmap**: Visual progression through phases
