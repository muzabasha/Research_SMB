┌──────────────────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING GUIDE                                                       │
└──────────────────────────────────────────────────────────────────────────────┘

   If Build Fails:
   1. Check Vercel build logs
   2. Look for specific error message
   3. Compare with expected output in docs
   4. Verify files were pushed correctly
   5. Clear Vercel cache and retry
   
   If Pages Don't Load:
   1. Check browser console for errors
   2. Verify URL is correct
   3. Check network tab for failed requests
   4. Try hard refresh (Ctrl+Shift+R)
   5. Clear browser cache
   
   If Assessments Don't Work:
   1. Check console for JavaScript errors
   2. Verify assessments-data.ts is correct
   3. Check stage ID matches
   4. Verify state management working
   
   If Navigation Broken:
   1. Check Link components
   2. Verify routes are correct
   3. Check for JavaScript errors
   4. Verify Next.js routing working
