## 🛠️ Helper Scripts

### git-commit-push.bat
**Purpose**: Automated git push script  
**Use When**: You want one-click deployment  
**How to Use**: Double-click the file  
**What It Does**: Adds, commits, and pushes all changes  
**Platform**: Windows

### test-build.bat
**Purpose**: Local build test script  
**Use When**: You want to test before pushing  
**How to Use**: Double-click the file  
**What It Does**: Runs `npm run build` and shows results  
**Platform**: Windows
